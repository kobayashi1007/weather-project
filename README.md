# Weather Project

台灣天氣預報查詢網站

## 功能特色

- 🌤️ 查詢台灣各縣市天氣預報
- 🗺️ 互動式台灣地圖，顯示選中城市位置
- 🎨 現代化 UI 設計
- 📱 響應式設計，支援各種裝置

## 使用技術

- HTML5
- CSS3
- JavaScript (ES6+)
- Bootstrap 5
- 中央氣象局開放資料 API

## 使用方式

1. 在搜尋欄輸入縣市名稱（例如：臺北市）
2. 或點擊城市便捷框快速查詢
3. 查看該城市的天氣預報資訊

## API

使用中央氣象局開放資料平台 API：
- API 端點：F-C0032-001（36小時天氣預報）
